Qus: Is Arthamedha better than Rich Dad Poor Dad by Robert Kiyosaki

Ans: Yes absolutely. There are many differences between Rich Dad Poor Dad and Arthamedha, but the main difference is that Rich Dad Poor Dad talks about assets to become rich, but does not teach how to create or buy assets. Whereas Arthamedha teaches in detail the principles of wealth creation and management, by learning which one can easily create or buy an asset. So to become rich, Arthamedha guides better than Rich Dad Poor Dad.

To be rich you must have three things desire, knowledge and action. If you have desire and action than you need knowledge.
